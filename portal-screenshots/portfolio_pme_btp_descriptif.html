Contexte client

PME du secteur BTP victime d'une tentative d'intrusion sur Microsoft 365. Aucune politique MFA en place, accès non gouvernés, connexions depuis l'étranger non bloquées. Besoin : sécuriser le tenant Azure en urgence et livrer une infrastructure reproductible.

Livrables réalisés

Entra ID

3 utilisateurs nominatifs + 2 groupes de sécurité (Leaders / Employees) — conformité RGPD, traçabilité des accès

RBAC

Rôles Contributor / Reader / Billing Reader assignés par profil métier — principe du moindre privilège

MFA

Conditional Access CA-01 — MFA obligatoire pour tous les utilisateurs, bloque 99% des attaques par credential stuffing

Géo-blocage

Conditional Access CA-02 — toutes connexions hors France bloquées, Named Location configurée

IaC

6 fichiers Terraform versionnés sur GitHub — infrastructure reproductible en 2 minutes, audit trail complet

Stack technique

Terraform Azure CLI Microsoft Entra ID Conditional Access Git / GitHub VSCode
